BasicSetTheory

發表於 2023-09-16 更新於 2023-09-19 分類於 概率論 閱讀次數：

def. 定義，prop. 性質，con. 概念，ex. 例子

Sets

def. Let \(\Omega\) be a set, then \(\mathcal{P}(\Omega)\) is the collection of all subsets of \(\Omega\). \(\mathcal{P}\) is called power set.

ex.

\[ A=\{x, y, z\} \]

\[ \mathcal{P}(A) = \{ \{x\}, \{y\}, \{z\}, \{x, y\}, \{y, z\}, \{x, z\}, \{x, y, z\}, \{\} \} \]

den. Collection of subsets of A (not all subsets, probably) is denoted as \(\mathcal{A}\)

con. intersection over arbitrary index sets

\[ \begin{aligned} \left[0,10\right] &= [0,1]\cup [0,2]\cup \cdots\cup [0,10]\\ &=\bigcup_{n=1}^{10}[0,n]\\ &=\bigcup_{n\in \{1,2,\cdots,10\}}[0,n] \end{aligned} \]

\(\{1,2,\cdots,10\}\)就是一個 index set

prop.

\[ 1_A\leqslant 1_B\Leftrightarrow A\subset B \]

def.

\[ \liminf_{n\rightarrow\infty}A_n=\bigcup_{n=1}^{\infty}\bigcap_{k=n}^{\infty}A_k \]

理解角度一：換成邏輯語言

\[ \omega\in \bigcup_{n=1}^{\infty}\bigcap_{k=n}^{\infty}A_k \]

\[ \Rightarrow \exists n_0\ s.t.\ \omega\in \bigcap_{k=n_0}^{\infty}A_k \]

\[ \Rightarrow \omega\in A_k\ \forall k\geqslant n_0 \]

ex.

\[ \limsup_{n\rightarrow\infty}A_n=\bigcap_{n=1}^{\infty}\bigcup_{k=n}^{\infty}A_k \]

try: \(\omega \in \bigcap_{n=1}^{\infty}\bigcup_{k=n}^{\infty}A_k\Rightarrow \omega\in \bigcup_{k=n}^{\infty}A_k\ \forall n\Rightarrow \omega\in A_k\ \exists k\geqslant n\)

理解角度二：函數列收斂點

\[ 收斂：\forall \varepsilon>0\ \bigg [ \exists N\ \forall n>N \ |f_n(x)-f(x)|<\varepsilon \bigg ] \]

\[ \bigcup_{N=1}^{\infty}\bigcap_{n=N}^{\infty}\{x:|f_n(x)-f(x)|<\varepsilon \}=\liminf_{N\rightarrow\infty}C_N \]

\[ 發散：\exists \varepsilon_0>0\ \forall N \ \exists n_0>N \ s.t. \ |f_{n_0}(x)-f(x)|\geqslant \varepsilon_0 \]

\[ \bigcap_{N=1}^{\infty}\bigcup_{n=N}^{\infty}\{x:|f_{n_0}(x)-f(x)|\geqslant \varepsilon_0\}=\limsup_{N\rightarrow\infty}D_N \]

理解角度三：頻率角度

happens eventually

\[ \liminf_{n\rightarrow\infty}A_n=\{\omega:\omega\in A_k\ \forall k\geqslant n_0\} \]

happens infinitely often

\[ \limsup_{n\rightarrow\infty}A_n=\{\omega:\omega \in A_{k_n} \ \exists k_n\geqslant n\ \forall n\} \]

Algebra

def.

Let \(\mathcal{A}\) be collection of subsets of \(\Omega\), \(\mathcal{A}\) is closed under \(\big[\bigcup,\bigcap,(\cdot)^c,\limsup,\liminf\big]\) if the set obtained by performing [operation] on \(\mathcal{A}\) is still on \(\mathcal{A}\)

def.

An algebra/field \(\mathcal{A}\) is a non-empty class(=collection) of subsets of \(\Omega\), s.t.

  1. \(\Omega\in \mathcal{A}\)
  2. \(A\in\mathcal{A}\) implies \(A^c\in\mathcal{A}\)
  3. \(A,B\in \mathcal{A}\) implies \(A\cup B\in\mathcal{A}\) \(\Rightarrow\) finite union

2+3:

\[ (\bigcup_{k=1}^nA_k^c )^c=\bigcap_{k=1}^nA_k\in\mathcal{A} \]

def.

An algebra is called a \(\sigma\)-algebra if

\(\{A_n\}_{n=1}^{\infty}\subset \mathcal{A}\) implies \(\bigcup_{n=1}^{\infty}A_n\in \mathcal{A}\)

\[ 同時(\bigcup_{n=1}^{\infty}A_n^c)^c=\bigcap_{n=1}^{\infty}A_n \]

相當於把algebra的第三個條件延拓到countable上

正例. \(\mathcal{B}=\mathcal{P}(\Omega), \mathcal{B}=\{\varnothing,\Omega\}\)

類比有限維線性空間

def.

\(\mathcal{A}\) is a collection of subsets of \(\Omega\), \(\sigma(\mathcal{A})\) is a \(\sigma\)-algebra generated by \(\mathcal{A}\), satisfying

  1. \(\mathcal{A}\subset \sigma(\mathcal{A})\)
  2. \(\mathcal{B}\) is another \(\sigma\)-algebra containing \(\mathcal{A}\), we have \(\sigma(\mathcal{A})\subset \mathcal{B}\)

def.

Borel \(\sigma\)-algebra

\[ \mathcal{B}_{\mathbb{R}}=\sigma(\text{open subsets of } \mathbb{R}) \]

實變結論：任何 \(\mathbb{R}\) 上的 open set 都是 countable、disjoint 個開區間的 \(\bigcup\)

直觀的說，subsets \(\Rightarrow\) \(\bigcup_i (a_i,b_i)\)。實際上有

\[ H_1=\{(a,b):-\infty\leqslant a\leqslant b\leqslant \infty\} \]

\[ 則 \ \sigma(H_1)=\mathcal{B}_{\mathbb{R}} \]

Borel \(\sigma\)-algebra 中的集合叫 Borel sets

基本技巧：\(A=(A\backslash B)\bigcup (A\bigcap B)\)
