特殊微分方程的解

發表於 2023-07-31 更新於 2023-08-21 分類於 ODE 閱讀次數：

Separable Variables

当\(x,y\)可分离时， \[ Q(x,y)\frac{dy}{dx}+P(x,y)=0\tag{6.6} \] \[ P(x,y)dx+Q(x,y)dy=0\tag{6.61} \]

\[ \Rightarrow f(x)dx+g(y)dy=0\tag{6.62} \] \[ \Rightarrow \int f(x)dx+\int g(y)dy=C\tag{6.63} \]

Def 7.1 Let \(z=f(x,y)\) define \(z\) as a function of \(x\) and \(y\) in a region \(R\). The function \(f(x,y)\) is said to be homogeneous of order n if it can be written as

\[ f(x,y)=x^ng(u)\tag{7.11} \]

where \(u=\dfrac yx\) and \(g(u)\) is a function of \(u\); or alternately if it can be written as

\[ f(x,y)=y^nh(u)\tag{7.11} \]

where \(u=\dfrac xy\) and \(h(u)\) is a function of \(u\)

也可定义为：A function \(f(x,y)\) is said to be homogeneous of order n if \[f(tx,ty)=t^nf(x,y)\tag{7.16}\]

Homogeneous Coefficients

Definition 7.2. The differential equation \[P(x,y)dx+Q(x,y)dy=0\tag{7.3}\] where \(P(x,y)\) and \(Q(x,y)\) are each homogeneous functions of order \(n\) is called a first order differential equation with homogeneous coefficients.

Theorem 7.32. If the coefficients in \((7.3)\) are each homogeneous function of order \(n\), then the substitution in it of \((7.31),(7.33)\) will lead to an equation in which the variables are separable.

\[y=ux\ \Leftarrow \ \left(u=\frac{y}{x}\right),\quad dy=udx+xdu,\quad \tag{7.31}\]

\[x=uy\ \Leftarrow \ \left(u=\frac{x}{y}\right),\quad dx=udy+ydu,\quad \tag{7.33}\]

Linear Coefficients

\[ x=\bar{x}+h,\quad y=\bar{y}+k\tag{8.11} \] \[ \Rightarrow \bar{x}=x-h,\quad \bar{y}=y-k\tag{8.12} \] 解线性系数的微分方程， \[ (a_1x+b_1y+c_1)dx+(a_2x+b_2y+c_2)dy=0\tag{8.2} \]

两条直线不平行

Method 1

两条直线不平行，因而相交，设交点为\((h,k)\)，是新的原点\((h,k)=(\bar{0},\bar{0})\)。把\((8.11)\)替换进\((8.2)\)，由交点的性质，常数项可忽略。 \[ a_1h+b_1k+c_1=0 \] \[ a_2h+b_2k+c_2=0 \]

转换成homogenous的形式 \[ (a_1\bar{x}+b_1\bar{y})d\bar{x}+(a_2\bar{x}+b_2\bar{y})d\bar{y}=0\tag{8.24} \]

Method 2

换元 \[ u=a_1x+b_1y+c_1\tag{8.3} \] \[ v=a_2x+b_2y+c_2 \] 微分 \[ du=a_1dx+b_1dy\tag{8.31} \] \[ dv=a_2dx+b_2dy \] 解方程，用\(du,dv\)表示\(dx,dy\)，代入\((8.2)\)式，再用homogenous的方法。

两条直线平行

Method 1失效。只能换元，都用\(u\)表示 \[ (2x+3y-1)dx+(4x+6y+2)dy=0 \] \[ u=2x+3y-1, \quad \Rightarrow\quad 4x+6y+2=2u+4 \] 然后就回到\(u\)和\(x\)或\(y\)组成的homogenous形式。

Exact DE

  1. Riemann integral \[ \lim_{n\rightarrow \infty}\sum_{k=1}^nf(x_k)\Delta x_k=\int_a^bf(x)dx \]

  2. 微积分基本定理 \[ F(x)=\int_{x_0}^xf(u)du \tag{9.13} \]

\[ \rightarrow F'(x)=f(x), \quad a<x<b\tag{9.14} \]

  1. 要让 \[\int_{x_0}^xP(x,y)dx+\int_{y_0}^yQ(x,y)dy\tag{9.16}\] 有定义，必须\([x_0,x]\times [y_0,y]\subset D\)

Definition 9.19. A region is called a simply connected region if every simple closed curve lying entirely in the region encloses only points of the region.

shrink closed curve to a point without non-domain points getting on the way

其实就是单连通域，对应的是多连通域

要让\((9.16)\)有定义

  1. 共同定义域是S.C. region \(R\)
  2. \((x_0,y_0)\in R\)
  3. 长方形\([x_0,x]\times [y_0,y]\subset R\)

Def 9.23 \[ P(x,y)dx+Q(x,y)dy\tag{9.24} \] is called an exact differential if it is the total differential of a function \(f(x,y)\). \[ P(x,y)=\frac{\partial}{\partial x}f(x,y)\quad Q(x,y)=\frac{\partial}{\partial y}f(x,y)\tag{9.241} \]

Def 9.27 \[ P(x,y)dx+Q(x,y)dy\tag{9.28}=0 \] is called exact differential equation if follows \((9.241)\).

Theorem 9.3. A necessary and sufficient condition that the differential equation \[ P(x,y)dx+Q(x,y)dy\tag{9.31}=0 \] be exact is that \[ \frac{\partial}{\partial y}P(x,y)=\frac{\partial}{\partial x}Q(x,y\tag{9.32}) \]

解法一：

套公式 \(\displaystyle{f(x,y)=\int_{x_0}^x P(x,y)dx+\int_{y_0}^y Q(x_0,y)dy}=c\)

或\(\displaystyle{f(x,y)=\int_{y_0}^yQ(x,y)dy+\int_{x_0}^xP(x,y_0)dx}=c\)

解法二： \[ P(x,y)dx+Q(x,y)dy=0 \] By \(\displaystyle{\frac{\partial}{\partial x}f(x,y)=P(x,y)}\), \[ f(x,y)=\int P(x,y)dx+R(y)\tag{$\star$} \] 再由 \[ \frac{\partial}{\partial y}f(x,y)=Q(x,y) \] \[ \underbrace{\frac{\partial}{\partial y} \int P(x,y)dx}_{\displaystyle{\int \frac{\partial}{\partial y}P(x,y)dx}}+R'(y)=Q(x,y) \] 解出\(R(y)\)再代回\((\star)\)式.

Integrating Factor

Def 10.5 A multiplying factor which converts an inexact diff eq into an exact one is called integrating factor.

\[ \frac{\partial}{\partial y}[hP(x,y)]=\frac{\partial}{\partial x}[hQ(x,y)]\tag{10.62 $\star$} \]

  1. \(h=h(x)\)

\[ \frac{dh(x)}{h(x)}=\frac{\frac{\partial}{\partial y}P(x,y)-\frac{\partial}{\partial x}Q(x,y)}{Q(x,y)}dx\tag{10.64} \]

更特殊地，\(F(x)=\displaystyle{\frac{\frac{\partial}{\partial y}P(x,y)-\frac{\partial}{\partial x}Q(x,y)}{Q(x,y)}}\)，所以\(\displaystyle{h(x)=e^{\int F(x)dx}}\)

  1. \(h=h(y)\)

\[ \frac{dh(y)}{h(y)}=\frac{\frac{\partial}{\partial x}Q(x,y)-\frac{\partial}{\partial y}P(x,y)}{P(x,y)}dy\tag{10.68} \]

  1. \(h=h(u), u=xy\)

\[ \frac{dh(u)}{h(u)}=\frac{\frac{\partial}{\partial y}P(x,y)-\frac{\partial}{\partial x}Q(x,y)}{yQ(x,y)-xP(x,y)}du\tag{10.72} \]

  1. \(h=h(u),u=\frac xy\) \[ \frac{dh(u)}{h(u)}=\frac{y^2\left[\frac{\partial}{\partial y}P(x,y)-\frac{\partial}{\partial x}Q(x,y)\right]}{xP(x,y)+yQ(x,y)}du\tag{10.76} \]

  2. \(h=h(u),u=\frac yx\)

\[ \frac{dh(u)}{h(u)}=\frac{x^2\left[\frac{\partial}{\partial x}Q(x,y)-\frac{\partial}{\partial y}P(x,y)\right]}{xP(x,y)+yQ(x,y)}du\tag{10.76} \]

Linear DE

Def 11.1 A linear diff eq of the first order is \[ \frac{dy}{dx}+P(x)y=Q(x)\tag{11.11} \]

Integrating factor is in the form \(\displaystyle{e^{\int P(x)dx}}\), by \((9.32)\).

\[ \left[P(x)e^{\int P(x)dx}y-Q(x)e^{\int P(x)dx}\right]dx+e^{\int P(x)dx}dy=0\tag{11.13} \]

\[ \underbrace{e^{\int P(x)dx}dy+P(x)e^{\int P(x)dx}ydx}_{d(e^{\int P(x)dx}y)}=Q(x)e^{\int P(x)dx}dx\tag{11.16} \]

\[ e^{\int P(x)dx}dy=\int Q(x)e^{\int P(x)dx}dx+C\tag{11.19} \]

This is a true general solution of \((11.11)\). \[ y=e^{-\int P(x)dx}\int Q(x)e^{\int P(x)dx}dx+Ce^{-\int P(x)dx}\tag{11.191} \]

Bernoulli Equation

\[ \frac{dy}{dx}+P(x)y=Q(x)y^n\tag{11.5} \]

两边同时乘以\((1-n)y^{-n}\) \[ (1-n)y^{-n}\frac{dy}{dx}+(1-n)P(x)y^{1-n}=(1-n)Q(x)\tag{11.52} \] \[ \frac{d}{dx}(\underbrace{y^{1-n}}_{u})+(1-n)P(x) \cdot \underbrace{y^{1-n}}_{u}=(1-n)Q(x)\tag{11.53} \] \(u=y^{1-n}\)，转化为线性DE的形式 \[ u'+(1-n)P(x) \cdot u=(1-n)Q(x)\tag{11.53} \]
