Definition 2.68.

A set in the plane is called a region if it satisfies the following two conditions:

  1. Each point of the set is the center of a circle whose entire interior consists of points of the set.
  2. Every two points of the set can be joined by a curve (not a line) which consists entirely of points of the set.

Definition 2.81.

The relation
$$
f(x,y)=0\tag{2.82}
$$
defines $y$ as an implicit function of $x$ on an interval $I:a<x<b$, if there exists a function $g(x)$ defined on $I$ such that
$$
f[x,g(x)]=0\tag{2.83}
$$
for every $x$ in $I$.

Definition 3.4.

F[x,f(x),f’(x),\cdots,f^{(n)}(x)]=0\tag{3.42}

Definition 3.6.

A relation $f(x,y)=0$ will be called an implicit solution of the differential equation

$$F(x,y,y’,\cdots,y^{(n)})=0 \tag{3.61}$$

on an interval $I:a<x<b$, if

  1. it defines $y$ as an implicit function of $x$ on $I$,i.e., if there exists a function $g(x)$ defined on $I$ such that $f[x,g(x)]=0$ for every $x$ in $I$, and if

  2. $g(x)$ satisfies 3.61, i.e., if

$$
F[x,g(x),g’(x),\cdots,g^{(n)}(x)]=0 \tag{3.62}
$$

for every $x$ in $I$.

Definition 4.3.

The functions defined by

$$
y=f(x,c_1,c_2,\cdots,c_n)\tag{4.31}
$$

of the $n+1$ variables, $x,c_1,c_2,\cdots,c_n$ will be called an n-parameter family of solutions of the $n$th order differential equation

$$F(x,y,y’,\cdots,y^{(n)})=0\tag{4.32}$$

if $\forall c_i(i=1,\cdots,n)$, the resulting function $f(x)$ defined by $(4.31)$ satisfies $(4.32)$. That is

$$F(x,f,f’,\cdots,f^{(n)})=0\tag{4.33}$$

$n$ order $\nLeftrightarrow$ $n$ parameter solutions

$\nRightarrow$: $(y’’)^2+y^2=0$

$\nLeftarrow$: $(y’-y)(y’-2y)=0\leftarrow (y-c_1e^x)(y-c_2e^{2x})=0$

Definition 5.12.

If $y=f(x)$ or $f(x,y)=0$ defines $y$ as a function of $x$ which satisfies
$$y’=F(x,y), \quad a<x<b\tag{5.11}$$
then the graph of this function is called an integral curve, it is the graph of a function which is the solution of $(5.11)$.

Remind

$$y’=F(x,y)\Leftrightarrow \text{Direction field}\tag{ODE}$$

$$y(x_0)=y_0 \Leftrightarrow \text{Point}\ (x_0,y_0)\tag{I.C.}$$

$$y(x) \Leftrightarrow \text{Integral curve}\tag{Solution}$$

Definition 5.4.

An ordinary point of (5.11) lies on one and only one of its integral curves.

Definition 5.41.

A singular point of (5.11) meets two requirements

  1. NOT ordinary point: no/more than one integral curve
  2. at least one ordinary point $o_0\in \cup(s_0,\varepsilon),\forall \varepsilon$

Definition 6.13.

Let $y=f(x)$ define $y$ as a function of $x$ on an interval $I$. The differential of y, written as $dy$ (or $df$) is defined by
$$
(dy)(x,\Delta x)=f’(x)\Delta x\tag{6.14}
$$

Theorem 6.2.

If $y=x$, then
$$(dy)(x,\Delta x)=(dx)(x,\Delta x)=\Delta x\tag{6.22}$$

$dy$,$dx$ are differentials, not increment. $y,x$ after $d$ are functions, not independent variables

Theorem 6.3.

If $y=f(x)$ defines $y$ as a function of $x$ and $x=g(t)$, $y=f[g(t)]=F(t)$, define $x$ and $y$ as functions of $t$, then
$$(dy)(t,\Delta t)=f’[g(t)][g’(t)(dt)(t,\Delta t)]=f’(x)(dx)(t,\Delta t)\tag{6.31}$$

Definition 6.4.

For $z=f(x,y)$,
$$(dz)(x,y,\Delta x,\Delta y)=\frac{\partial f(x,y)}{\partial x}\Delta x+\frac{\partial f(x,y)}{\partial y}\Delta y\tag{6.41}$$

Theorem 6.5.

For $z=f(x,y)$, $x=x(r,s,\cdots)$,$y=y(r,s,\cdots)$
$$dz=\frac{\partial f(x,y)}{\partial x}\Delta x+\frac{\partial f(x,y)}{\partial y}\Delta y\tag{6.52}$$

Definition 7.1.

Let $z=f(x,y)$ define $z$ as a function of $x$ and $y$ in a region $R$. The function $f(x,y)$ is said to be homogeneous of order n if it can be written as

$$
f(x,y)=x^ng(u)\tag{7.11}
$$

where $u=\dfrac yx$ and $g(u)$ is a function of $u$; or alternately if it can be written as

$$
f(x,y)=y^nh(u)\tag{7.11}
$$

where $u=\dfrac xy$ and $h(u)$ is a function of $u$

也可定义为：A function $f(x,y)$ is said to be homogeneous of order n if $$f(tx,ty)=t^nf(x,y)\tag{7.16}$$

Homogeneous Coefficients

Definition 7.2. The differential equation
$$P(x,y)dx+Q(x,y)dy=0\tag{7.3}$$
where $P(x,y)$ and $Q(x,y)$ are each homogeneous functions of order $n$ is called a first order differential equation with homogeneous coefficients.

Theorem 7.32.

If the coefficients in $(7.3)$ are each homogeneous function of order $n$, then the substitution in it of $(7.31),(7.33)$ will lead to an equation in which the variables are separable.

$$y=ux\ \Leftarrow \ \left(u=\frac{y}{x}\right),\quad dy=udx+xdu,\quad \tag{7.31}$$

$$x=uy\ \Leftarrow \ \left(u=\frac{x}{y}\right),\quad dx=udy+ydu,\quad \tag{7.33}$$
