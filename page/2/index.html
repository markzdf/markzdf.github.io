解决hexo本地和部署不一致问题

發表於 2023-07-31 更新於 2023-08-20 分類於 開發

这个问题困扰我很久，为什么hexo s跟hexo d结果不一样？起初我以为是deploy要慢一点，但第二天早上醒来还是原样。

后来找到原因。点进网站对应的Github界面，可以看到默认的branch是master。改成别的分支就能解决问题。

点击分支，点击 View all branches，点击右上角的 New branch。

新建之后返回刚才的界面，并将 master 改成刚建的分支 main。

接着在 Settings 的 Default branch 一栏点击$\rightleftarrows$，把默认分支改为 main，点击 Update ，点击红色的 I understand。

最后再找到左边的导航栏点击Pages，在Branch栏下，同样把 master 改为 main，保存。

最后记得在博客文件夹的 _config.yml 内将deploy部分的branch改为main。

即可实现部署和本地一致。

龙虾卷超好吃，底下面包烤的很香，龙虾也是超多超新鲜。必须要在这边买房了，这样天天能吃到。

7.3

听Magin的话把social network换成了financial derivatives。这门课在系统上显示是special topics in finance，还以为是水课就没选，讲衍生品就太好了，正好下学期学金融工程会用到很多。这老师目前看上去还不错，他说拿到A就能拿他的推荐信，他要么不写、要么就写优质的那种，还挺不错。第一天上课很轻松，虽说有的同学看起来不轻松，看来这两年还是学了点东西的。

每个班都有大约一半中国人，处境比较尴尬。一方面现在已经tired of跟中国人打交道，但欧洲人又没办法融入进去。今早先是认识了俩Italian，聊的还好，后面又来一个German，他融入Italians毫无困难，话题多很能聊。从足球到Vegas赌博，再到clubbing和Spanish girl，再到玩射击，推荐这推荐那，真的很会玩。反思一点就是，不要站那什么都不说，想点话说（虽然这不容易）。还有说的前提是听懂，我听不懂，说的好也没用。需要练听力（平时对话就在练）和熟悉地理（欧洲+美国）。

下午去gym健身+打了会篮球，晚上回Cheney又跟隔壁打了会桌球，现在有点纠结的是怎样平衡学习和玩的时间。学习又包括学金融、数学、托福、kaggle，玩包括社交、健身、还有没探索到的玩法，真是太忙。可以周一到周四努力学，玩只包括健身，周五到周日去周边城市逛。

躺在床上又想起些细节。早上去吃饭见人就说good morning。平时再会后说have a nice day/you too。下公交要对司机大声说thank you。多说thanks没坏处。一老板娘说sorry I haven’t heard you in first hand超快。早上behavior课有个Berkeley的cs学生很自信、很会聊、很聪明，真的优秀。下午和一个学ds的老美打球，各有胜负。这边老黑对中国人的态度会比白人好点，但也看场合。

7.4

今天独立日，去三藩玩。美国人都长的挺好看。玩了scooter，贵，爽，傻逼。三小孩骑自行车唱着什么东西，一个朝消防车司机喊thank you! 一个脱手骑车，一个前轮起飞。喝了个tripology，恶心，像蘸蟹脚的酱料。去了fisherman’s wharf，小龙虾很不喜欢就没久待。看了golden gate bridge，没电影里好看。晚上六七点快给冻死，打了个uber送到地铁站回来。七点多地铁站就准备开始百鬼夜行了。

本来去三藩是想看firework，结果等不及他太晚，但也玩到了。听小龙虾描述他们的课很高强度，每节课/每周有作业不说，还有discussion和project。
